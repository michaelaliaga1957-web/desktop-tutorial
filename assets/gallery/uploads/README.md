# Drop new photos here

Upload your detailing photos into this folder (any name, JPG/PNG/HEIC/WebP).
Claude will crop, color-grade, compress, handle privacy (plates/bystanders),
and place the best ones into the website gallery.

**Best shots for a high-end look:**
- Finished paint in **open shade** (not direct sun) — reflections read as luxury
- Wheels/tires after dressing
- A clean dashboard / interior surface, evenly lit
- True before → after pairs of the *same* area
